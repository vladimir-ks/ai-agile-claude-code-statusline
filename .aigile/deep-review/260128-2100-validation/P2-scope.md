## Critical Focus
**Setup/Installation UX:**
- Can a new user ACTUALLY install this by following docs?
- Are prerequisites clearly stated?
- Is the configuration step obvious?
- What happens if they skip a step?

**Troubleshooting Flow:**
- Do error messages map to troubleshooting guide?
- Are diagnostic steps effective?
- Can user self-diagnose common issues?
- Are there missing troubleshooting scenarios?

**Operational Documentation:**
- How to upgrade safely?
- How to monitor for issues?
- How to roll back if broken?
- Maintenance requirements clear?

**User Mental Model:**
- Do docs explain HOW it works (not just what)?
- Are data source priorities clear?
- Is cache behavior understandable?
- Can user predict behavior?
